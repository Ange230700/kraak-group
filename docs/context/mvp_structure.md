# Point 2 — Structure du MVP du site web KRAAK

## 1. Objectif de la structure 🧭

La structure du MVP doit permettre aux visiteurs de **comprendre rapidement qui
est KRAAK, ce que l’organisation propose et comment passer à l’action**.

L’architecture du site doit être :

- simple ;
- claire ;
- adaptée au mobile ;
- orientée conversion.

## 2. Structure globale recommandée 🌐

Le MVP peut être construit autour de **5 pages principales** :

1. **Accueil**
2. **À propos**
3. **Services**
4. **Programmes / Formations**
5. **Contact**

Cette structure est suffisante pour lancer une première version professionnelle, tout en restant facile à maintenir.

## 3. Rôle de chaque page 📄

### 1) Accueil

Page d’entrée principale du site.  
Elle doit :

- présenter KRAAK en quelques phrases claires ;
- mettre en avant la mission, la proposition de valeur et les domaines d’intervention ;
- orienter rapidement vers les services, les programmes ou la prise de contact ;
- inclure des appels à l’action visibles.

### 2) À propos

Page dédiée à l’identité de l’organisation.  
Elle doit présenter :

- la vision ;
- la mission ;
- les valeurs ;
- la crédibilité de KRAAK et son positionnement.

### 3) Services

Page qui explique de façon structurée les trois pôles d’activité :

- **Formation**
- **Gestion de projet**
- **Conseil en immigration**

Chaque service doit comporter une description simple, les bénéficiaires visés et l’action attendue (contact, demande, intérêt).

### 4) Programmes / Formations

Page destinée à détailler les offres ou parcours proposés.  
Elle peut inclure :

- les types de formations ;
- les publics concernés ;
- les bénéfices attendus ;
- un bouton pour demander plus d’informations ou rejoindre une liste d’attente.

### 5) Contact

Page de conversion principale.  
Elle doit contenir :

- un formulaire simple ;
- les coordonnées utiles ;
- éventuellement un lien WhatsApp, email ou prise de rendez-vous.

## 4. Navigation principale 🔗

Le menu principal du MVP doit rester court et clair :

- Accueil
- À propos
- Services
- Programmes
- Contact

Un bouton d’action visible dans l’en-tête peut être ajouté :
**Demander une consultation** ou **Nous contacter**

## 5. Éléments de conversion à prévoir ✅

Pour soutenir les objectifs du MVP, les appels à l’action doivent être visibles à plusieurs endroits du site :

- sur la page d’accueil ;
- à la fin de chaque section service ;
- sur la page Programmes / Formations ;
- sur la page Contact.

Les principales actions à encourager sont :

- contacter KRAAK ;
- demander une consultation ou une formation ;
- manifester son intérêt pour un programme.

## 6. Résultat attendu

Une structure web simple, crédible et efficace, qui permet de lancer rapidement
la présence en ligne de KRAAK, tout en préparant l’ajout futur de nouvelles
pages, fonctionnalités ou programmes.
